# Data
Dans le dossier Data, vous retrouverez les feature_cards, le data_card.yaml, le csv de la matrice de corrélation, ainsi que les fichiers csv qui constitue le dataset.

# Modèles
Vous retrouverez trois notebooks, "Traitement_visualisation_datacard.ipynb", "modele_language.ipnyb" et "modele_math.ipnyb".

# Traitement_visualisation_datacard.ipynb
Traitement, affichage des données, statistiques, graphs, feature cards, datacards.

# modele_language.ipnyb
Entrainement du modèle language mean score, cross validation, optimisation bayésienne, métrics, visualisation graphique, export du modèle, model cards.

# modele_math.ipnyb
Entrainement du modèle math score, cross validation, optimisation bayésienne, métrics, visualisation graphique, export du modèle, model cards.

# /artifacts
Dans le dossier artifacts, vous retrouverez les exports des deux modèles, ainsi que les model cards au format markdown.

# requirements.txt
Pour faciliser l'installation des dépendances nécessaires, vous retrouverez un fichier requirements.txt avec les librairies nécessaires.
